# e621 downloader

e621 downloader - программа для скачивания постов, коллекций и избранных пользователей на платформе e621.net

## Скачиваниe

Скачайте rar архив из последнего релиза и пользуйтесь!

## Как работает программа

Программа работает на API e621 [(Пример)](https://e621.net/posts.json?tags=meow_skulls%20hyper%20-intersex&limit=100) и через данный API происходит скачивание постов. После чего в папке, где используется программа, создается отдельная папка, где складываются посты.
