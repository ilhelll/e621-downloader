# e621 downloader

e621 downloader - программа для скачивания постов, коллекций и избранных пользователей на платформе e621.net
e621 downloader - a program for downloading posts, collections, and favorite users on the e621.net platform.

## Скачиваниe/Downloading

Скачайте rar архив из [последнего релиза](https://github.com/ilhelll/e621-downloader/releases/tag/e621) и пользуйтесь!
Download the rar archive from [the latest release](https://github.com/ilhelll/e621-downloader/releases/tag/e621) and enjoy!

## Как работает программа/How the program works

Программа работает на API e621 [(Пример)](https://e621.net/posts.json?tags=meow_skulls%20hyper%20-intersex&limit=100) и через данный API происходит скачивание постов. После чего в папке, где используется программа, создается отдельная папка, где складываются посты.
The program runs on the e621 API [(Example)](https://e621.net/posts.json?tags=meow_skulls%20hyper%20-intersex&limit=100) and downloads posts through this API. After that, a separate folder is created in the folder where the program is used, where the posts are stored.
